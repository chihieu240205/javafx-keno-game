# CS342 Project 2 — JavaFX Keno Game

A desktop Keno game built with **JavaFX + Maven** for CS342.

## Overview
This project implements a GUI-based Keno-style game application using Java.  
It demonstrates:
- JavaFX UI design
- Event-driven programming
- Game logic implementation
- Maven project structure
- Basic object-oriented design

## Project Structure
- `src/` — Java source code and resources
- `pom.xml` — Maven build configuration
- `README.md` — project overview
- Project documents (report / wireframe / class diagram PDFs)

## Features
- Interactive JavaFX user interface
- Keno game board and user selections
- Game state updates and result display
- Modular project structure for maintainability

## Tech Stack
Java, JavaFX, Maven
